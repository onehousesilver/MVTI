## 👀 Recommendation Algorithm

* 내 MBTI를 기반으로 한 영화 추천 알고리즘

1. 나의 MBTI랑 같은 MBTI를 가진 유저가 영화를 '내 리스트'에 추가하면 리스트에 많이 담긴 순서대로 영화를 추천
2. 만약 '스파이더맨'을 'ISFP'유저들이 많이 리스트에 담은 경우, 다른 ISFP 유저에게 추천해줌(가중치를 둠)
